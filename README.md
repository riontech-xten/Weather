# Weather
This repository democracy use of Android Jetpack component and how we can reduce the dependancy injection boilerplate code using dagger-hilt. 

- Kotlin Coroutines
- Kotlin Flow
- Room Database
- ViewModel
- LiveData
- Dagger-Hilt
- Retrofit
- Databinding
- Navigation UI 
- Navigation Safeargs
- Weather api http://api.openweathermap.org
- Unit Test
